Using extensions to make your code look cleaner and proper formating


August 21, 2021

There are are many number of extensions present in vs code that you can use but today i'll tell you may favourite extensions

1. ESLint :- It is one of the popular extensions you will find in vs code it quicky analysis your code to find if there is any problem like syntax error or missing semi-colon of any bug it quickly give a red error line there itself and also provide a fix option to fix that code automatically

2. Prettier :- It is another extension you will find in vs code it quicky formate your code with proper space and proper line break

3. Bracket Pair Colorizer :- It is another beautiful extension present in vs code it give a matching colour to opening and closing bracket which make developer easy to know which bracket belongs together

4. Live Server :- It is another one of the best extension provided by vs code it launches a local development server for both static as well ad dynamic pages with a live reload. everytime you'll save your code you will see the changes reflected in the browser without killing and restarting your server because of this extension you can easyliy spot error and can do quick experiments with your code
